Effective translation workflow

CLI tool for effectively update translation keys, edit translations
and open translation interface.

npm install get-translation -g

Workflow timeline

Write your source code

var properties = {
title: gt(’LANDING_TITLE’)
};

Update translation keys

$ gt update

Get latest translation keys

$ gt log
@1 LANDING__TITLE | The bes...
@2 LANDING__DESCRIPTION | H...
@3 BASE__LOG_OUT | Log out
@4 BASE__SIGN_IN | Sign in
...

Edit translations

$ gt set @1 “Some title”

Compile translations

$ gt compile
